### From-Scratch Transformer for Translation
**English-French neural machine translation**

Complete Transformer architecture built from the ground up without pre-trained models:
- Custom multi-head attention mechanisms
- Positional encoding implementation
- Encoder-decoder architecture
- Metal GPU acceleration
- BLEU score evaluation

**Performance:** 67.26% test accuracy, 0.52 BLEU score  
**Technologies:** Python, TensorFlow, Keras
